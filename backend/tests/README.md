# DevsuApi Test

A Minimal API Test built with .NET 9, ready to run, test, and deploy with Docker.

## Prerequisites

- [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0) (Preview/RC)
- [Docker](https://www.docker.com/get-started)

**Local Setup:**

1. Clone the repository:  
   `git clone https://github.com/yourusername/devsu_test_full_stack_BP.git`
2. Navigate to the project directory:  
   `cd devsu_test_full_stack_BP/backend/tests/DevsuApi.UnitTests`
3. Restore dependencies:  
   `dotnet test --logger:"console;verbosity=detailed"`
